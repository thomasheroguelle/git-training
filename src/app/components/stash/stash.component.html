How to stash ?

La commande $ git stash prend vos changements non commités (stagés et non stagés), les enregistre pour une utilisation ultérieure, puis les replace dans votre copie de travail.

delete a stash : $ git stash pop

Voir sa liste de stash: $ git stash list

Donner un nom à son stash: $ git stash -m "test stash"

Voir les modifications apportées au stash : $ git stash show

Afficher les modifications complètes au stash : $ git stash show -p

Créer une branche avec son stash : $ git stash branch
