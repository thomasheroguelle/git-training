Commandes générales :

Afficher les changements dans un commit spécifique : $ git show ID_du_commit

Supprimer un fichier suivi par Git : $ rm mon_fichier

Afficher l'historique des commits avec un format condensé : $ git log --oneline

Afficher l'historique graphique :

$ git log --graph --oneline --all
