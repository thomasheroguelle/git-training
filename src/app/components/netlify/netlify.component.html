To deploy an existing project on Netlify :

On the project :

In the project , create a file to the project root named netlify.toml

Add in the file :

[[redirects]]
from = "/*"
to = "/index.html"
status = 200

Deploy the project with :

  • ng build
  • npm run build

On Netlify :

Add New Site > Import An Existing Project

Select the project with GitHub
