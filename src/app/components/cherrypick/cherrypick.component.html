How to CherryPick ?

User Case :

J'ai créé une branche qui ne me convient plus , je souhaite récupérer mon travail sur cette branche puis la supprimer :

1. $ git checkout -b myNewBranche

2. $ git cherry-pick hashDuCommit(de la old-branch)

Note1 : on peut cherry-pick le nombre de commit qu'on souhaite, je peux très bien faire plusieurs git cherry-pick hashDuCommit(de la old-branch)

Note2 : le add s'effectue seul, il n'y a plus qu'à push
