How to CherryPick ?

User Case :

(On admet que je n'ai qu'un commit) : J'ai créé une branche qui ne me convient plus , je souhaite récupérer mon travail sur cette branche puis la supprimer :

1. git checkout -b myNewBranche

2. git cherry-pick hashDuCommit(de la old-branch)

3. git commit, git add, git push
