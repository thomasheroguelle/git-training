How to push ?

1. Go to IntelliJ

2. Add manually for each files with '+'

3. Enter the name of the commit ex: [SCOP-588]: Delete component files . Then, click on 'Commit'

4. Click on "Push", the "pre-commit" in the folder must be commented

5. Then, do $ git checkout .

6. $ git checkout develop

7. $ git pull develop and git fp

8. Then, go back to your branch

9. Do $ git rebase origin and git push -f or git fp
